Fermi is a statically typed systems programming language designed for simplicity, predictability, and efficient native code generation.

The language and compiler are built with a focus on clean architecture and long-term maintainability.

> **Warning**
>
> Fermi is currently in pre-alpha. The language, compiler, and tooling are under active development and may change without notice.

## Getting Started

### Requirements

- C11-compatible compiler (`clang` recommended)
- LLVM development tools (`llvm-config`)

Documentation for the language, compiler, and development process is available in the `docs/` directory.
